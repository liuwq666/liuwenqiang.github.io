# 程序员作品集网站

这是一个专业的程序员作品集网站，包含了所有大厂、面试官、HR最爱看的6大核心模块。

## 核心模块

1. **顶部导航栏** - 固定在页面顶部，包含首页、关于我、技能、项目、联系我
2. **首页大屏** - 3秒抓住面试官眼球，展示你的名字、职业和简短介绍
3. **关于我** - 展示你的背景、学习经历、性格和优势
4. **技能栈** - 分为编程语言和工具/框架两类，一目了然
5. **项目展示** - 面试官最看重的部分，展示你能做东西
6. **联系我 + 简历下载** - 方便面试官联系你

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标库

## 如何使用

1. 克隆或下载本项目到本地
2. 打开 `index.html` 文件，修改以下内容：
   - 导航栏链接
   - 首页大屏的名字、职业和介绍
   - 关于我的内容
   - 技能栈列表
   - 项目展示的内容和链接
   - 联系信息和简历下载链接
3. 保存修改后，直接在浏览器中打开 `index.html` 文件查看效果

## 部署到 GitHub Pages

1. 注册 GitHub 账号
2. 新建一个仓库
3. 把网站文件上传到仓库
4. 开启 Pages 功能
5. 获得一个网址，例如：`你的名字.github.io`

## 优化建议

1. **统一字体** - 保持网站字体一致，看起来更专业
2. **调整颜色** - 主颜色建议选择蓝色/紫色，最适合程序员
3. **响应式设计** - 确保手机打开也能正常显示
4. **内容优化** - 确保所有项目链接能打开，简历能下载，文字没有错别字，图片不模糊
